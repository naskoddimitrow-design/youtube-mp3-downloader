Download YouTube videos as MP3 easily, with **automatic updates**!

## Features

- Download YouTube videos in MP3 format  
- Choose download folder  
- Adjustable MP3 quality: 128, 192, 256, 320 kbps  
- Dark / Light mode with matching startup animation  
- Download history with easy management  
- Settings panel for theme and quality  
- **Automatic updater**: checks GitHub Releases for new EXE versions  

## How to Use

1. Run `youtube_mp3_downloader.exe`  
2. Paste YouTube links in the box  
3. Select a download folder  
4. Click **Download MP3**  
5. Optionally, check your **History** or change settings  

## Automatic Updates

- The app checks GitHub for the latest release at startup  
- If a new version is available, a popup will ask if you want to update  
- Clicking **Yes** downloads the new EXE and restarts the app automatically
